Features

GitHub Profile Lookup

Fetch public GitHub profile data instantly using the GitHub REST API. Just enter a username and get:


  • Bio and profile image
  • Repository list with direct links
  • Followers, following, and more

Clean, Intuitive Navigation

Designed for clarity and ease of use. Whether you're exploring one profile or many, the layout keeps things simple and focused.

Dynamic Color Styling

Using Color Thief, the app extracts the dominant color from a user's GitHub profile image and applies it to the UI. To ensure visual clarity and personality, the app intelligently detects when the dominant color is a neutral gray and instead uses a more vibrant secondary color. This creates a personalized, expressive experience for each search, while avoiding washed-out tones that clash with the light layout.

Insights Dashboard

After each search, the app generates insights based on the retrieved data:


  • Repository count
  • Most starred repo
  • Follower/following ratio
  • Account age and activity

Search History

The Insights page also includes a history of past searches, so you can revisit profiles without retyping usernames. Great for comparison or tracking.

Privacy-Friendly

No login required. No data stored. All information is fetched directly from GitHub's public API and displayed locally.
